# Quotes Scraper

A Python web scraping project built using Requests, BeautifulSoup, and SQLite3.

# Features
   Extract quotes and authors from website
   Pagination scraping
   Time delay handling
   Custom request headers
   Session handling using requests.Session()
   SQLite database integration
   Data storage instead of simple CSV files

# Technologies Used
   Python
   Requests
   BeautifulSoup4
   SQLite3

# Learning Outcomes
  This project helped me understand:
     HTTP requests
     Web scraping workflow
     Session management
     Database integration
     Responsible scraping practices
